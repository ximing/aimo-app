**Learnings for future iterations:**
  - Services must be registered in app/_layout.tsx using `register()` from @rabjs/react before they can be used with `useService()`
  - Components using services should be wrapped with `view()` decorator to make them reactive
  - Theme colors use hex codes (e.g., #007AFF for blue, #FF9500 for orange)
  - Empty data should return null to hide the component entirely
  - For navigation in cards, use `router.push('/memos/[id]')` pattern
